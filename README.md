| 🧭 **[MCP-Travel-Agent](https://github.com/zain-the-npc/MCP-Travel-Agent)** | Multi-agent trip planner — LangGraph orchestrates a researcher + budget agent, pulling live flight/hotel/place data via 4 independent MCP servers | `LangGraph` `MCP` `GPT-4o-mini` `FastAPI` `React` |

| 💻 **[npc-ai](https://github.com/zain-the-npc/npc-ai)** | Fully offline, self-hosted AI assistant — local LLM via Ollama, zero cloud dependency | `FastAPI` `React` `Ollama` |
